# Spectroscopic Study of 48Ti

Author: Dikshit Majee  
MSc Physics Thesis

## Overview
This repository contains my MSc thesis on gamma-ray spectroscopy of 48Ti.

The work focuses on nuclear structure analysis using:

- RDCO (Directional Correlation from Oriented states)
- IPDCO (Integrated Polarization Directional Correlation)

## Experimental Setup

The experiment was performed using the INGA detector array.
